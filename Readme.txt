Instructions for Compilation:
GCC: g++ -O3 client.cpp -o client
GCC: g++ -O3 server.cpp -o server
Clang-LLVM: clang++ -O3 client.cpp -o client
Clang-LLVM: clang++ -O3 server.cpp -o server

Testing:
GCC: g++ -O3 -Wall -W -pedantic-errors client.cpp -o client
GCC: g++ -O3 -Wall -W -pedantic-errors server.cpp -o server
Clang-LLVM: clang++ -O3 -Wall -W -pedantic-errors client.cpp -o client
Clang-LLVM: clang++ -O3 -Wall -W -pedantic-errors server.cpp -o server

Instructions for Using the Program:
Launch the server executable from a host machine, then clients can use their
executable to communicate with the server.
